# OMPL Sample
OMPLのサンプルを実行してみるだけのレポジトリだな  
ソースコードは以下のリンクを参考  
[http://robotics.naist.jp/edu/text/?Robotics%2FOMPL#OMPLProgramming](http://robotics.naist.jp/edu/text/?Robotics%2FOMPL#OMPLProgramming)

## プログラムの実行
各プログラムのディレクトリに移動して

```bash
mkdir build
cd build
cmake ..
make
./実行ファイル名
```
